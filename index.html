  • L’objectif de ce projet était de créer un programme de classification de petits articles appelés « dépêches ».
    Ce programme devait être codé en Java sous l’environnement de développement Intelij.
    Il devait, automatiquement, associer à chaque dépêche une catégorie, à l’aide d’un algorithme bien précis.

    Compétences techniques mises en œuvre :
    • • Création d’ArrayList de Paires
    • • Association d’un score à une dépêche en fonction d’un lexique
    • • Lecture de fichier texte dans une fonction
    • • Tri et classement d’ArrayList de Paires


    Compétences transversales mise en œuvre :
    • • Travail d’équipe sur un projet de programmation
    • • Synthétiser un projet à l’oral
    • • Recherches personnelles pour enrichir un code
